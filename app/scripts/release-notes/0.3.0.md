A design pass, and one setting you asked for.

**New**
- **Appearance**: choose System, Light or Dark in Settings › General. System is the default.
- **How this works**: the "?" next to the title explains what Lid Boot actually changes, that it's reversible, and that it only applies from a full shutdown.
- **Copy Diagnostics** in Settings › About — your Mac model, macOS version and the current setting, for bug reports.

**Improved**
- Follows the macOS 26 look: the window uses the system material instead of painting its own background, and all text now uses the system's type scale.
- The app icon sits properly alongside Apple's own icons.
- The menu bar popover is tidier: one button, and a gear for everything else.
- Fewer permanent warnings — the detail moved into "How this works".

**Fixed**
- Lid Boot no longer refuses to run on MacBooks newer than the M1. It was checking the model name, and Apple stopped using "MacBook…" identifiers in 2022, so an M2 Air or an M4 was told it wasn't a laptop. It now asks the hardware whether it has a lid.

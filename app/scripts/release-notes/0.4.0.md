LidBoot is now open source — and this release moves everything into one place.

**New**
- **Open source**: the full source is now public at github.com/h3x4d3x4/LidBoot, MIT-licensed. For an app that runs commands as root, you shouldn't have to take the safety claims on faith.
- The app icon shown in the window and About box is now the real app icon, not a drawing of it.

**Changed**
- One name everywhere: **LidBoot**, joined.
- Updates and downloads now come from the main repository. **If you're on 0.3.1 or earlier, this is the last update the old feed will offer** — install this one and you're on the new feed automatically.
- Support link is now Buy Me a Coffee.

**Improved**
- Offline-proof installs: the app itself now carries its notarization ticket, so the very first launch works with no internet connection.

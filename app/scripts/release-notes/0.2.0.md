First beta.

- Two switches: stop your Mac starting up when you open the lid, connect power, or both.
- Reads the current setting without asking for a password; only changes need one.
- Restore Default puts your Mac back exactly as it shipped.
- Copy Terminal Command gives you the equivalent `nvram` line for scripting.
- English and Portuguese.

Requires an Apple silicon MacBook on macOS 15 (Sequoia) or later.

Note: this only affects starting up from a full shutdown — waking a sleeping Mac
is unaffected, and pressing a key or touching the trackpad still starts it up.
